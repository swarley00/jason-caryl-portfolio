Marvel API Hack

For this project, the idea was to take an existing API and create a self-directed project around it. This served as a way to practice using AJAX. I chose the Marvel API, an API that I found to be not as well documented or maintained as I had originally hoped, to work with. The site I made searches for your favorite Marvel character then displays the 20 most recent comics they appear in.
